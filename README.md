# Flappy bird - Phaser

The Flappy Bird game created with [Phaser](https://phaser.io/) framework.

## Technologies used
- [Phaser](https://phaser.io/)
- [TypeScript](https://www.typescriptlang.org/)
- [Webpack](https://webpack.js.org/)

## How to run
For build
```sh
npm run dev
```

or start localy
```sh
npm run start
```
